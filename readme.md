# Petrol Fever
Jeśteśmy ignacym (wynalazcą ropy naftowej) i wydobywamy ropę.
Widok izometryczny.
Gameplay loop:
- Znajdujesz ropę pod ziemią.
- Stawiasz wiertło
- podłączasz wiertło rurami do ropy
- wydobywasz
- sprzedajesz u jednej z firm.
